SQLIO_ResultsToExcel.ps1 processes all SQLIO output files in the SQLIO_ResultFilesSQLIO_ResultFiles directory and consolidates into an Excel workbook.

Parameters:
- Drive (e.g. "E", without quotation marks)

Hardcoded in SQLIO_ResultsToExcel.ps1:
- $InputDirectory - directory containing output from SQLIO

To run:
In Windows explorer; right-click file and and choose "Run with Powershell"

Files and Directories:
- SQLIO_ResultsToExcel.ps1
- SQLIO_SampleChart.xlsx (sample processed data and chart)
- SQLIO_ResultFiles (directory with sample  SQLIO output files)
